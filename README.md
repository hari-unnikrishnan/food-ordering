# Restaurant POS / Food Ordering Dashboard

This project includes:
- **Frontend**: React
- **Backend**: Django + Django REST Framework
- **Database**: SQLite (default)

## Layout
- Left sidebar: Categories
- Center: Food cards grid
- Top: Search bar
- Right sidebar: Cart / Order details
